In the participant-fixed-effects regression using the continuous abstractness score, the association between abstractness level and left frontal ROI HbO response was not statistically significant (beta=1.311e-08, SE=1.251e-08, p=0.303503, 95% CI [-1.249e-08, 3.870e-08]). This suggests that replacing the binary label with a continuous abstractness measure does not by itself yield robust evidence for a frontal abstractness effect in the current dataset. The binary benchmark remained non-significant (p=0.209749), the fixed-window continuous benchmark remained non-significant (p=0.271618), and the no-override continuous model remained non-significant (p=0.481741).
