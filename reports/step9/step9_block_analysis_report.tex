\section*{Overview}
This report implements the Step 9 block-adjusted analysis using the same Step 7 participant-session cohort, yielding 36 participant-sessions, 30 questions, and 1075 valid trials.
The realized block metadata were merged from the cleaned behavioral table, so all diagnostics and models use the actual session order rather than an idealized task template.
Descriptively, the valid-trial counts by early blocks began as block 1: 39 abstract trials, block 1: 69 concrete trials, block 2: 39 abstract trials, block 2: 68 concrete trials.
